🎮 Mystery Doors Adventure Game
A simple text‑based adventure game written in Python.
The player chooses between doors and boxes to find treasure or face surprises.
This project is written in simple English to make it easy for beginners to understand and learn from.
----------------------------------------
🚀 Features
Text‑based interactive gameplay

Multiple choices and different outcomes

Beginner‑friendly Python code

Uses basic input, conditions, and ASCII art

Great for learning decision‑making in Python
----------------------------------------
🧠 How the Game Works
The player starts on an island

Two doors appear: red and blue

Choosing the red door leads to three boxes

Each box has a different surprise

Choosing the blue door ends the game

Any wrong input shows an “Invalid choice” message
----------------------------------------
📂 File Name
mystery_doors_game.py
----------------------------------------
▶️ How to Run
Make sure Python is installed

Run the game: python mystery_doors_game.py
----------------------------------------
